##### 魔方的转动规则
魔方的转动以面为单位,举例来说:
U 表示顶面按顺时针转动90度，容易理解U<span style="color: red;">'</span>表示按逆时针方向转动90度,U2 表示顶面转动180度,也就是2个U,其他面以此类推.
假定是用右手在转动魔方,那么U就表示顶层正转,U'表示顶层反转.下图列出了常用的魔方转动方式,这些基础的符号需要牢记,因为他们将是后面公式中的组成部分.
